# small-chat-room

## Built With

* Express (Framework used)
* Handlebars (View Engine)
* Socket.io
* Mongoose
* @hapi/joi (For request validation)
* Bootstrap 4/ moment.js/ Jquery etc.
* The profile pictures are stored in a s3 bucket.
